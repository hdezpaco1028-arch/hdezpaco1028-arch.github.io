How to replace placeholders & swap to exact product links (quick)

  1. Upload product images to your GitHub repo: Repo → Add file → Upload files → Commit.
  2. Open the uploaded image, click Raw, copy raw URL (looks like https://raw.githubusercontent.com/USERNAME/REPO/main/image.jpg).
  3. Edit index.html and in the products array (near the top of <script>) set the product's img value to that raw URL.
  4. To replace Buy links: set product.amazon to the exact Amazon product page URL (or add your affiliate tag).
